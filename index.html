Rock,Paper and Scissor

For my portfolio, I developed an engaging Rock, Paper, Scissors project utilizing HTML, CSS, and JavaScript. This interactive game offers a classic experience with a modern twist, showcasing my proficiency in front-end web development. Users can click on their choice of rock, paper, or scissors, and the game dynamically displays the outcome against the computer's selection. With sleek and intuitive design elements crafted through CSS, the project provides an enjoyable user experience across various devices. JavaScript handles the logic behind the game, ensuring seamless gameplay and accurate result calculation. This project not only demonstrates my technical skills but also highlights my ability to create fun and interactive web applications.
